- [x] Implementação de Payment Tracking para monitoramento de checkout.
- [x] Suporte dinâmico para programas: Initial, COS, Transfer, EB-2 (2 etapas) e EB-3 (6 etapas).
- [x] Métricas de abandono de 24h e conversão no Dashboard Administrativo.
- [x] Implementação do Dashboard Unificado (Admin/Operations).
- [x] Portal do Vendedor (Analytics, Links, Leads).
