Sempre que um PROMPT tiver o nome TASK no início ou no meio dele, adicione-o no relatório de atividades de hoje (RELATORIO_TECNICO_YYYY_MM_DD.md).

Regras Gerais:
- Responda sempre em PT-BR.
- Prefira soluções simples e escaláveis.
- Evite duplicação de código.
- Mantenha os arquivos entre 200-300 linhas de código.
- Adicione logs se em Modo Depurador.
- Use Modo Planejador para mudanças complexas.
